概述 关于《五笔加加》原版。
　　五笔加加是由北京六合源软件技术有限公司开发的一款以五笔输入为主的输入法软件，其突出的特点是兼容性和稳定性极佳，易用性好，简洁而小巧，受到许多用户喜爱。可惜该软件早已停止开发，它的最后一个版本是“五笔加加1.0试用版”。
关于《五笔加加Plus》
　　针对五笔加加原版的一些不足和错误之处，Easycode在原版的基础上对其进行了改进和修正，为与原版区别，将此改进版命名为“五笔加加Plus”。《五笔加加Plus》的出现并非要取代原版，只把它看作是原版之外的另一个选择，希望对一部分人“有点用”。
关于《五笔加加Plus For IBus》
　　参照五笔加加及五笔加加Plus的一些功能设计，通过对ibus-table源代码的修改，实现了基于ibus的具有五笔加加风格的输入法。
使用这个版本是免费的吗？
　　当然，这是一个基于LGPL-2.1许可证发布的开源输入法，其源代码发布于 https://github.com/yanzilisan183/ibus-wbjj，可以在符合LGPL-2.1的情况下使用、修改和再发布。
