五笔加加Plus For IBus ◆常见问题 ◆符号列表 ◆快速入门
快 速 入 门
|快速选择重码字词
  1. 用空格键选首位重码、左Shift键选第二重码，右Shift键选第三重码；
  2. 在“首选项”中启用<;><'>键选第二、三重码。
|字符集切换
  1. 按“Ctrl＋;”组合键依次切换；
  2. 单击语言栏中的“字符集”图标；
  3. 在“首选项”中选择所需的字符集。
|中文/英文切换
  1. 按Ctrl键或Shift键切换中英文输入状态(可在“首选项”中设定热键)；
  2. 单击语言栏中的“模式”图标。
|中/英文标点切换
  1. 按“Ctrl＋.”组合键；
  2. 单击语言栏中的“标点”图标；
  3. 在“首选项”中选择所需的标点模式。
|全角/半角切换
  1. 按“Shift＋空格”组合键；
  2. 单击语言栏中的“字符宽度”图标；
  3. 在“首选项”中选择所需的字符宽度。
|输入数字
  1. 只需通过大键盘直接输入阿拉伯数字，即可从提示窗口中获得其它格式的数字，如键入123，则提示窗口中将显示：１２３、一百二十三、壹佰贰拾叁元整等各种形式的数字。然后用空格及左右Shift键或Ctrl＋数字选择备选。
  2. 若用户想输入阿拉伯数字时直接上屏，可使用小键盘输入数字，或关闭中文数字输入功能（在“首选项”中取消对“中文数字”的选择）。
特别提示：当用户输入中文数字时，须通过空格及左右Shift键或Ctrl＋数字选择备选，若使用数字键，则视为正在输入数字。
|输入日期
在启用“中文数字”状态下，通过大键盘以“年.月.日”或“年-月-日”或“年/月/日”的形式输入阿拉伯数字表示的日期，如输入2020.3.15，提示窗口中即可显示2020年3月15日, ２０２０年３月１５日, 二〇二〇年三月十五日, ２０２０．３．１５ 四种日期格式。通过空格及左右Shift键或Ctrl＋数字选择备选。
|输入特殊符号（单个符号）
输入相应符号中文名称的五笔编码即可输入该符号。例如：输入“顿号”的编码gbkg，即可得到“、”。可参见《五笔加加特殊符号列表》
|输入系列符号（成组符号）
输入相应系列符号中文名称的五笔编码，然后从提示窗口选择即可。超过3个的重码用数字键选择。例如：输入“平假名”的五笔编码gwqk，即可获得日文平假名。可参见《五笔加加系列符号列表》
|重复输入最近的字词或符号
“Z＋空格”（对以编码方式打出的标点符号有效）
提示：Z 键同时提供最近输入字词编码及编码反查功能。
|实时造词
  1. 首先依次输入构成新词的单字/词(拼音五笔均可)；
  2. 接着按“Ctrl＋=”组合键进入造词状态，提示窗口显示用户最新输入的两个单字；
  3. 通过按 =、- 两个键可增减构成词组的字；
  4. 通过Backspace键可删除待造词提示行中的第一个字，按Delete键可删除待造词提示行中的第二个字;
  5. 按空格(或回车)将新词加入五笔词库;或按左Shift(或2键)将新词加入拼音词库。
特别提示：在造词过程中，用户可随时按Esc键退出造词状态。
|删除词组
  1. 当候选提示窗口中出现词组时，直接按“Alt＋序号”组合键快捷删除。
|自动调频
在“首选项”中启用“动态调频”。
|手动调频
尚未实现
|回车键作用
回车键用于使编码字母上屏。
|翻页键设置
  1. PageDown和PageUp(总是可用)
  2. 单击提示框的翻页按钮(总是可用)
  3. Tab和Shift＋Tab
  4. 方向键（备选列表横向时↑↓用于翻页，备选列表竖向时←→用于翻页）
  5. =和-
  6. .和,
|词库管理
参考“首选项”对话框“词库”选项卡中提供的功能。
|有用的快捷键
  • “Ctrl + Alt + Shift + F1” 打开帮助文档
  • “Ctrl + Alt + Shift + F2” 打开首选项对话框
  • “Ctrl + Alt + Shift + BackSpace” 重新启动IBus守护进程(对于输入法图标不显示时很有用)
|拼音输入、反查五笔编码
首先按 Z 键，进入拼音辅助输入功能。用户可接着输入某字的全拼，通过翻页可找到所需的汉字，同时系统在每个字后面标出该字的五笔编码。该功能不仅可帮助用户输入不会拆分的字，还可起到反查编码的作用。
